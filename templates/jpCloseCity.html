What's up?

  • This will accept an input list of Japanese zip codes and find the closest city for you.
  • The coordinates and names for cities are pulled from http://weathernews.jp/pollen/xml/obs.xml
  • And zip codes are geocoded (turned into coordinates) using OpenStreet Map
  • Note: Requests are throttled at 1/sec per OpenStreet Maps requirements

Geocoder data provided by OpenStreet Map. See https://operations.osmfoundation.org/policies/nominatim/ For usage notes, licensing, and restrictions.

Provide a file with one zip code or city name per line.
